#### Hello World!

Try this [code](http://play.golang.org/p/OccSs5jC9Y):

```go
package main

import "fmt"

func main() {
	fmt.Println("Hello World!")
}
```

You can either:

- `go run hello/main.go`
- `cd hello/` and `go build` and `./hello`
- `cd hello/` and `go install` and `hello`
